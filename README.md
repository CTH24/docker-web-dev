## Installation

### Release

```bash
unzip =(wget -qO- https://github.com/CTH24/web-dev-docker/archive/refs/tags/v1.1.3.zip) \
  && mv web-dev-docker-1.1.3 project-name
```
## Commands

```bash
./start
```

Complete system reset. All Files and Databases will be deleted.

```bash
./reset
```
